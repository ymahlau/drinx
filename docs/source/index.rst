Drinx Documentation: Datclass Registry in JAX
==========================================================================================

Installation
------------

You can install drinx simply via pip:

.. code-block:: bash

   pip install drinx

If you want to use the GPU-acceleration from JAX, you can install afterwards:

.. code-block:: bash

   pip install jax[cuda]
